# SW동행 데모데이 팀프로젝트

## 소개

전국 습지 모니터링과 실시간 수질 데이터를 시각화, 지역별 현황을 제공하는 웹 애플리케이션입니다.

## 기능

- kakao map api를 통해 지도에 습지 위치
- python matplotlib를 이용한 연도별 기온 변화 그래프

## 사용기술

- Express
- React
- kakao map API

## 개발 파트

- BACKEND: Express, kakao map API
- [github 바로가기](https://github.com/doca41531/expressdemoday)
